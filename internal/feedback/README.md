# Feedback

Internal feedback, critique, and outside-reading artifacts live here. This
folder is intentionally not copied into the public site artifact.

- `Human/` - human critique, Reddit feedback, and application-facing feedback
  records.
- `model/` - quarantined model-generated readings and benchmark-like stress
  tests.
- `quarantine/` - high-risk, adversarial, speculative, or scratch feedback that
  should not be promoted without a separate review pass.

Public docs may summarize accepted lessons from these files, but raw feedback
artifacts should stay internal unless a future pass explicitly promotes a
cleaned, evidence-tiered excerpt.
