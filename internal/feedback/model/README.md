# Model Feedback

This folder is the quarantine lane for model-generated readings, stress tests,
and benchmark-like interpretive reports.

The raw reports are useful as evidence about how outside systems misread,
compress, or overextend Sundog's public surface. They are not public claims
documents and should not be copied back into `docs/**` wholesale. Public docs
can summarize the lessons while preserving evidence tiers and current claim
boundaries.

- `Mythos-Benchmark.md` - outside-model conflation report and maintainer
  response.
- `gemini-benchmark.md` - outside Gemini read on site legibility, access
  ambiguity, and posture risks.
