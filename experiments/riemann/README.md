# Riemann Experiment Workspace

This folder is reserved for future `sundog_v_riemann` scripts, notebooks, and
small helpers.

Rules before adding executable work:

- start from the ledger in `docs/SUNDOG_V_RIEMANN.md`;
- freeze source, domain, thresholds, and output path in
  `docs/prereg/riemann/P0_DOMAIN_AND_RECEIPT_LOCK.md` or a later probe spec;
- write raw outputs under `results/riemann/<probe>/`;
- summarize reviewed outcomes under `docs/riemann/receipts/`;
- do not upgrade a receipt into an RH claim.

No executable probe has been added yet.
