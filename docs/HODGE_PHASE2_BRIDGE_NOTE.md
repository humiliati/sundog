## One-line Claim

Faraday and Aharonov-Bohm license a caution, not a method: local closure,
harmonic representatives, metric Hodge stars, and topological periods can be
beautifully structured shadows, but none of them constructs an algebraic cycle
or tests the Hodge conjecture.

## Why This Bridge Exists

The Hodge reader opened because the word "Hodge" was already doing real work in
the Faraday / Maxwell lane. Phase 8 uses the metric Hodge star `*` to state the
inhomogeneous Maxwell shadow `d*F = J`; Phase 7 isolates the Aharonov-Bohm
survivor as a harmonic / topological period. Those are legitimate electromagnetic
statements, with receipts.

The Hodge conjecture uses adjacent vocabulary from a different mathematical
world: Hodge decomposition, harmonic representatives, rational `(p,p)` classes,
and algebraic cycles on smooth projective complex varieties. The bridge is safe
only if it keeps those worlds aligned by register, not blended into one claim.

## The Safe Analogy

The permitted transfer is a discipline:

> Do not confuse the visible representative with the hidden body.

Faraday / AB taught the portfolio that local differential data can close exactly
while global structure survives in another register. Hodge is not the same
phenomenon, but it rhymes at the level of caution: a class can have the right
visible Hodge type while the missing object, if it exists, is an algebraic cycle.

This analogy is explanatory language. It is not a proof technique, a detector, a
simulation target, or a body-resistance measurement.

## Register Alignment

| Electromagnetic object | Tempting bad transfer | Safe Hodge-side reading |
| --- | --- | --- |
| `dF = 0` / Bianchi closure | "closed form means algebraic body" | Closedness or harmonicity is a statement about representatives/classes, not a cycle. |
| Metric Hodge star `*` in `d*F = J` | "Hodge-star machinery transfers to algebraic cycles" | The shared word names metric/cohomological structure; it supplies no algebraicity criterion. |
| Harmonic AB survivor / `H^1` period | "harmonic survivor equals rational Hodge class" | A topological period is not the rational `(p,p)` condition and not an algebraic cycle. |
| Loop holonomy `oint A = Phi` | "global shadow proves a hidden body" | It is an exact EM topological witness, bounded to the EM topology lane. |
| Cycle class map `cl(Z)` | "shadow-to-body map can be read off like holonomy" | The body-to-shadow direction is defined; the reverse direction is precisely Hodge. |

## Forbidden Transfers (`HODGE-PHYSICS-TRANSFER`)

This bridge fails if any later artifact says or implies:

1. electromagnetic Hodge-star calculations construct algebraic cycles;
2. a harmonic representative is the same thing as a rational Hodge class;
3. an Aharonov-Bohm topological survivor is evidence for any Hodge case;
4. Faraday's exact closure or Maxwell's dual-shadow closure is a Hodge argument;
5. a diagram of a differential form displays the algebraic cycle behind it;
6. Hodge is a regime-2 / body-resistance separation rather than an existence /
   algebraicity question.

If any of those appear, file `HODGE-PHYSICS-TRANSFER`, demote this bridge to an
internal warning paragraph, and do not proceed to a known-example gallery.
