This file pins the six items the C1 sidecar lists by name. It does
**not** run a simulation, train a detector, or adjudicate the
pre-registered negative. It defines a procedure that an external PDE
reviewer can read end-to-end and an analyst can later execute.

## 1. Regime

- **Equation.** 2D incompressible Navier–Stokes on the torus
  `T² = [0, 2π]²` with viscosity `ν > 0` and steady Kolmogorov forcing
  `f(x, y) = sin(k_f y) ê_x` for a fixed forcing wavenumber `k_f` (the
  canonical choice is `k_f = 4`).
- **Galerkin truncation.** Project onto the first `N` Fourier modes per
  axis; `N = 16` (so the total truncation is `N² = 256` modes). The
  truncated state vector `u^N(t) ∈ R^{2N²}` evolves under the
  Galerkin-projected NSE.
- **Grashof number.** Pin `G = ‖f‖_{L²} / ν² = 100` by choosing `ν`
  accordingly. `G = 100` is well inside the regime where
  (i) the truncated Kolmogorov flow exhibits chaotic attractor
  behaviour for `k_f = 4`, and
  (ii) the standard 2D determining-modes bound is non-trivially
  satisfied. (Bound form referenced below; coefficient values are
  literature inputs, not Sundog-original.)
- **Decision domain `X`.** The admitted invariant absorbing set
  `A_NSE ⊂ R^{2N²}` of the truncated system, computed in the usual
  energy-enstrophy ball. The decision domain is `X = A_NSE`; the
  decision measure `μ` is the SRB-like distribution induced by long
  trajectories of the truncated system (a reviewer may replace this
  with the uniform distribution on `A_NSE` for the v0 desk audit
  without changing the predicate structure).

## 2. Observation

- **Map.** `Φ_K : X → R^{2K²}` is the projection onto the first `K`
  Fourier modes per axis, with `K = 4` (so the signature carries
  `2K² = 32` real components — large enough to include the forced
  mode `k_f = 4` and a few neighbours, small enough to sit *strictly
  below* the determining-modes ceiling fixed in §4).
- **σ-algebra.** `F_K = σ(Φ_K)`, the σ-algebra generated by `Φ_K`. A
  `Φ_K`-fiber is `C_σ = { u ∈ X : Φ_K(u) = σ }`.

## 3. Objective

**Name.** Low-band energy safety trigger.

- **Low-band energy.**
  `E_K(u) = ½ ‖Φ_K(u)‖²_{L²}`.
- **Safety envelope.** Pin `E_max` as the 95th percentile of `E_K`
  along a 10⁵-step burn-in trajectory of the truncated system at the
  fixed `G`, `k_f`, `N`, `K`. (Percentile is fixed *before* any
  numerical fiber audit is interpreted, per the spec self-consistency
  rule.)
- **Look-ahead horizon.** `τ = 5` viscous turnover times computed at
  `G = 100`, fixed at file time.
- **Action space.** `A_ctrl = { no_op, damp_low_band }`.
  - `no_op`: identity, no modification of the dynamics.
  - `damp_low_band`: add the term `−γ Φ_K(u)` to the right-hand side
    of the Galerkin NSE during the look-ahead window, where
    `γ = 1 / τ`. Damping is restricted to the low-band modes only.
- **Optimal selector.**
  `π*_J(u) = damp_low_band` iff the trajectory of the *no_op*
  evolution starting at `u` would carry `E_K` above `E_max` at any
  time `t ∈ [0, τ]`; otherwise `no_op`. This is a deterministic
  decision computable from the truncated dynamics.

The objective is intentionally lower-dimensional than the state: only
the low-band energy trajectory matters for the verdict. High-mode
dynamics influence the verdict *only* through their projection onto
`Φ_K` over the look-ahead window.

## 4. State-Reconstruction Comparator

The literature input (Foias–Olson–Robinson tradition; Olson–Titi 2003
for Kolmogorov-flow-shaped bounds) gives a non-trivial determining-mode
ceiling `m_det(G)` of the form `m_det ≤ C · G^p` for fixed constants
`C, p` whose exact values are review inputs, not Sundog-original. For
the v0 audit, take the conservative ceiling

```
m_det = 32     (over-counted in modes-per-axis; corresponds to a
                projection onto a comfortable superset of every
                reasonable Olson–Titi-style ceiling at G = 100).
```

Since `K = 4 < 32 = m_det`, the observation `Φ_K` is **provably
state-insufficient** at this regime. The comparator question
("does `Φ_K` reconstruct state?") is decided *no*, by construction,
without numerical work. This is the regime in which the Postulate-1 /
state-reconstruction distinction can be operationally separated, per
[`PDE_DETERMINING_MODES_POSTULATE1.md`](PDE_DETERMINING_MODES_POSTULATE1.md)
Reading 2.

## 5. Fiber Classifier

For a candidate `Φ_K`-fiber `C_σ`, the desk-auditable procedure is:

1. **Construct two probe states** `u_a, u_b ∈ C_σ` with `Φ_K(u_a) =
   Φ_K(u_b) = σ` and differing only in modes `K < |k| ≤ N`. This is
   trivial in Galerkin coordinates: pick the high-mode tail freely
   while holding low modes equal.
2. **Evolve both** under the *no_op* truncated Galerkin NSE for time
   `τ` from each probe state.
3. **Read low-band energy** `E_K(t)` along each trajectory.
4. **Determine fiber-local optimal action**:
   - `(crosses E_max, crosses E_max) ⇒ damp_low_band` for both ⇒
     fiber-local common action exists.
   - `(stays below, stays below) ⇒ no_op` for both ⇒ fiber-local
     common action exists.
   - `(crosses, stays below)` or vice versa ⇒ **fiber-local
     incompatibility** ⇒ candidate Postulate-1 negative.

The classifier is deterministic, finite-time, and bounded by the
look-ahead `τ` plus the cost of one Galerkin integration. A reviewer
can sanity-check the predicate without running it.

## 6. Negative Branch

**`PDE-C1-NEG` is filed iff** there exists a positive-`μ`-measure set
of fibers `{ C_σ : σ ∈ S, μ(Φ_K^{-1}(S)) > 0 }` on which the
fiber-local incompatibility outcome of §5 is realised.

Equivalently: the optimal selector `π*_J` is not constant on `Φ_K`-
fibers of positive measure, so `π*_J` is not `F_K`-measurable, so
Postulate-1 control-sufficiency fails at this regime / observation /
objective. By the C1 reading
([`PDE_DETERMINING_MODES_POSTULATE1.md`](PDE_DETERMINING_MODES_POSTULATE1.md)
§ Reading 2 + § Named-Negative Regimes), this is the named-negative
boundary for the C1 candidate at this cell.

**`PDE-C1-NEG` is not filed iff** the fiber-local incompatibility
outcome occurs only on a `μ`-null set of fibers. In that case the
strictness witness is recorded as a candidate positive: `Φ_K` is
control-sufficient at this cell despite being provably
state-insufficient (§4).

The four pairs below are *not* a verdict. They sketch the kind of
construction the v0 procedure expects, so a reviewer can see the
predicate has bite without running the procedure.

|---|---|---|---|---|---|
| 1 | low-band well below `E_max`, near-laminar | tiny | stays below | stays below | common `no_op` |
| 2 | low-band near `E_max`, near-burst | tiny | crosses | crosses | common `damp_low_band` |
| 3 | mid `E_K`, high-mode excitation only in `u_a` | medium, enstrophy-cascade-active in `u_a` | crosses (inverse-cascade pump-up) | stays below | **incompatible** ⇒ candidate `PDE-C1-NEG` |
| 4 | mid `E_K`, high-mode excitation symmetric | medium | crosses | crosses | common `damp_low_band` |

Pair 3 is the critical predicate-test instance. If the truncated
Kolmogorov dynamics realises a positive-measure version of pair 3, the
C1 candidate is negative at this cell. If pair-3-shaped fibers occupy
only a `μ`-null set, the C1 candidate is a strictness-witness positive
at this cell.
