# Sundog High-Stakes Problem Matrix

Drafted: 2026-06-01

This is a target-selection ledger, not a claim ledger. It looks across the
current Sundog portfolio and asks which existing or new high-stakes problems
could support one of two honest outcomes:

1. a spectacle: a visual, executable, or public-facing workbench that makes the
   body/shadow problem legible; or
2. a boundary: a pre-registered null, vacuity, numerical wall, topology gate, or
   cost gate that teaches the next lane what not to claim.

Inputs read for this pass:

- `SUNDOG_V_GIMMICKS.md`: hidden-state game surfaces and anti-perfect-
  information framing.
- `SUNDOG_V_GRAVITY.md`: Goodhart sidestep, two-substrate convergence, ARC
  promotion, and structural-failure benchmarks.
- `CROSS_SUBSTRATE_NOTES.md`: the generality failure map and body-resistance
  vocabulary.
- `faraday/SUNDOG_V_FARADAY.md` plus the 2026-05-31 Faraday/Yang-Mills bridge:
  identity-success, Aharonov-Bohm exact topological separation, sourced Maxwell
  determinacy, and the abelian/non-abelian boundary.
- Recent math ledgers: Yang-Mills, Riemann, P-vs-NP, Navier-Stokes, ARC, and
  Cap-set / Unit-distance.

## Current Portfolio Reading

The important handoff from the Yang-Mills turn is not "try a bigger gauge
theory." It is:

> the same projection operator can be exact in an abelian/topological corner and
> non-separating in a non-abelian finite-lattice cell.

That makes the next target-selection rule sharper:

- Prefer problems where the obstruction is named before the run:
  cohomology, topology, incidence, trace formula, cost, local/global
  inconsistency, or numerical stiffness.
- Prefer problems where a clean null is publishable because it identifies the
  obstruction.
- Avoid "Sundog attacks famous problem" unless the first artifact is a bounded
  evaluator, toy model, finite-field analogue, or public workbench.

The live body-resistance axes are:

- **Dimensional resistance:** the body is too large for the shadow to reconstruct
  it. Cap-set and unit-distance have this as read-off mathematics. Control
  substrates have mostly been marginal so far, except low-dimensional synthetic
  chatv2.
- **Topological resistance:** local data are complete locally but miss a global
  period or holonomy. Aharonov-Bohm is the portfolio's exact earned witness.
- **Algebraic identity:** the shadow closes because the law is an identity.
  Faraday is the exact-zero anchor, not a regime-2 separation.
- **Non-separating null:** the invariant shadow is honest but carries no useful
  separating structure on the registered cell. Yang-Mills is the current
  example.

## Claim-Language Guardrails

Allowed:

> Sundog is using Hodge/Kakeya as stress tests for its projection discipline:
> can a bounded shadow expose where local, directional, or cohomological data
> does and does not determine the hidden body?

Allowed:

> A null is a result if it names the obstruction before the run.

Forbidden:

> Sundog is working toward a proof of Hodge, Kakeya, Yang-Mills, Riemann,
> Navier-Stokes, P-vs-NP, BSD, or Langlands.

Forbidden:

> A visual workbench is evidence for the unsolved problem.

Forbidden:

> A known finite-field theorem transfers to the Euclidean/open case without a
> separately registered bridge.
